# Pointer-Analysis-with-Polymorphism
A software used for analyzing pointers with polymorphism.

## How to clone the repo

```bash
git clone https://github.com/Iankulani/Pointer-Analysis-with-Polymorphism.git
cd Pointer-Analysis-with-Polymorphism
```

## How to run
```bash
python Pointer-Analysis-with-Polymorphism.py
 ```
